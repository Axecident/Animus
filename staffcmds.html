Staff Commands

Here are the staff commands :)

When warning someone type one of the commands listed below. This will give you a message that you then copy and send to the person (in DM) you are warning.

Warning Commands: When to use them

!w_profilepic = Use when someone has an inappropriate profile picture

!w_porn = Use when someone links/shares porn outside of the nsfw channel(if they're spamming, mute them first. Then send them the message.)

!w_ecchi = Use when someone links/shares ecchi material (if they're spamming, mute them first. Then send them the message.)

!w_loli = Use when someone links/shares loli/loli-like porn (if they're spamming, mute them first. Then send them the message.)

!w_spam = Use when someone is spamming

!w_racist = Use when someone is being racist towards a member in chat

!w_abuse = Use when someone is being abusive towards another member (and it doesn't seem like a joke)

!w_harass = Use when someone is harassing another member

!w_exploit = Use when someone is exploiting something they shouldn't be

!w_wrongstaff = Use when someone mentions a staff member higher than the lowest staff available (for example if they mention a Super Admin when there are moderators available)

!w_nodev = Use when someone asks for help from a developer instead of a staff member

 

If someone spams, use prune & mute on them.

Chat Management Commands:

?prune x (ex. ?prune 20) (Admin & Higher)

~prune x (ex. ~prune 14)

~mute @username (ex. ~mute @Charlie#0000)

-mute @username (ex. -mute @Charlie#0000) (incase ~mute isn't working)

~unmute @username (ex. ~unmute @Charlie#0000)

-unmute @username (ex. -unmute @Charlie#0000) (incase ~unmute isn't working)

 

If you see a command with (Admin & Higher) after it.

It means that only Admins & Super Admins can use those commands.
